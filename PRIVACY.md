# Privacy

This project does not collect, store, process, or share personal data.

- **No cookies** — no session cookies, no tracking cookies, no analytics cookies
- **No analytics** — no Google Analytics, no Plausible, no self-hosted analytics
- **No trackers** — no fingerprinting scripts, no third-party tracking pixels
- **No forms** — no contact forms, no newsletter signups, no data submissions
- **No first-party logs** — hosted on GitHub Pages, which may maintain standard access logs. This project has no access to or control over GitHub's infrastructure logs.

## Font Delivery

Typography is served from local static font files bundled with this site. Your browser does not need to contact Google Fonts or another font CDN to render the pages.

## Hosting

This site is hosted on **GitHub Pages**. Like any web host, GitHub necessarily receives standard HTTP request data to deliver pages, such as IP address, user agent, and requested URL. This project does not control GitHub's infrastructure logs. See [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).
