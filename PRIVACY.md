# Privacy

This site does not collect, store, process, or share any personal data.

- **No cookies** — no session cookies, no tracking cookies, no analytics cookies
- **No analytics** — no Google Analytics, no Plausible, no self-hosted analytics
- **No trackers** — no fingerprinting scripts, no third-party tracking pixels
- **No forms** — no contact forms, no newsletter signups, no data submissions
- **No server logs** — hosted on GitHub Pages, which may maintain standard access logs. This project has no access to or control over GitHub's infrastructure logs.

## External Services

The only external service used is the **Google Fonts CDN** for typography (JetBrains Mono, Source Serif 4, Source Sans 3, Special Elite, Cutive Mono). When your browser loads these fonts, Google may receive standard HTTP request data (IP address, user agent). This project has no control over Google's data handling. For details, see [Google Fonts Privacy](https://developers.google.com/fonts/faq/privacy).

## Hosting

This site is hosted on **GitHub Pages**. GitHub may collect technical information about page visits as part of their service. See [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).
