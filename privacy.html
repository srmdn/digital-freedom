Privacy

This site does not collect, store, process, or share any personal data.

No Tracking

There are no cookies, no analytics scripts, no fingerprinting, and no third-party tracking pixels on this site. You cannot be tracked because there is nothing to track you with.

No Data Collection

There are no contact forms, no newsletter signups, no user accounts, and no mechanism for submitting data. This is a static website — every page is a pre-built HTML file served directly.

External Services

The only external service used is the Google Fonts CDN for typography (JetBrains Mono, Source Serif 4, Source Sans 3, Special Elite, Cutive Mono). When your browser loads these fonts, Google may receive standard HTTP request data (IP address, user agent). This project has no control over Google's data handling. For details, see the Google Fonts Privacy FAQ.

Hosting

This site is hosted on GitHub Pages. GitHub may collect technical information about page visits as part of their service. See GitHub's Privacy Statement for details.
