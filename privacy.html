Privacy

This project does not collect, store, process, or share personal data.

No Tracking

There are no cookies, no analytics scripts, no fingerprinting code, and no third-party tracking pixels added by this project.

No Data Collection

There are no contact forms, no newsletter signups, no user accounts, and no mechanism for submitting data. This is a static website — every page is a pre-built HTML file served directly.

Font Delivery

Typography is served from local static font files bundled with this site. Your browser does not need to contact Google Fonts or another font CDN to render the pages.

Hosting

This site is hosted on GitHub Pages. Like any web host, GitHub necessarily receives standard HTTP request data to deliver pages, such as IP address, user agent, and requested URL. This project does not control GitHub's infrastructure logs. See GitHub's Privacy Statement for details.
