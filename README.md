# Digital Freedom Terminal

Interactive terminal-style narratives about digital freedom, privacy, and the erosion of personal liberties.

## About

A project that tells the story of digital freedom through an unconventional medium: the terminal itself. Each sequence is a self-contained HTML file that simulates a terminal session, revealing data, quotes, and arguments about the state of personal freedoms in the digital age.

No voiceover. No talking heads. Just text, timing, and tension.

## Philosophy

The terminal is where digital freedom is fought — in encryption, in open source, in self-hosted infrastructure, in the choice of what tools we trust with our data. Using the terminal as the storytelling medium is intentional: it's both the canvas and the message.

## Sequences

| File | Theme | Duration |
|------|-------|----------|
| `sequences/freedom-report.html` | The state of personal freedoms — data, quotes, call to action | ~60-90s |

Each sequence is a single HTML file with zero dependencies. Open it in a browser, click start, and screen record.

## Usage

Open any `sequences/*.html` file in a browser.
